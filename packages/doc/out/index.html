voidjs

Manage non-SPA pages with webpack and React

Guides

Why voidjs

voidjs is designed for managing non-SPA HTML pages. It brings a lot of benefits of webpack and React to non-SPA HTML pages developing.

  • Better management of assets with webpack
  • Manage html with Components, no more error-prone copy & paste
  • Pre-built static HTML

Use cases

  • Write HTML pages without JavaScript.
  • Write HTML pages with a few vanilla JavaScript.
  • Write HTML templates for server side languages other than Node.js, e.g., PHP.

System requirements

  • Node.js 12 or later
  • macOS, Windows and Linux are supported

Get started

  • To create a new voidjs project, run npm init:

    npm init voidjs ./my-app

  • Or if you are using yarn, run yarn create:

    yarn create voidjs ./my-app
