voidjs

Manage non-SPA pages with webpack and React

System requirements

  • Node.js 12 or later
  • macOS, Windows and Linux are supported

Use cases

  • Writing html pages without JavaScript.
  • Writing html pages with a few vanilla JavaScript.
  • Writing html templates for server side languages other than Node.js, e.g., PHP.

Get started

  • To create a new voidjs project, run npm init: 
    npm init voidjs ./my-app
  • Or if you are using yarn, run yarn create: 
    yarn create voidjs ./my-app

Development

Run npm run dev under your newly created project.

Production

Run npm run build under your project and voidjs will build the project to out directory.
